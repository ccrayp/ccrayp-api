Добро пожаловать на главную информационную страницу ccrayp-api.

Сервер
Данный API представляет из себя систему для наполнения сайта-визитки контентом. Реализован при помощи микрофреймворка Flask на языке программирования Python. Деплой призведен в облачном сервисе Render.
В качестве базы данных была использована PostgreSQL, развернутая в облачном сервисе Neon.
Реализованы эндпоинты для работы с контентом: добавление, редактирование, обновление и удаление данных.

Клиент
Для написания фронтенда: сайт-визитка, админ-панель, использовалась библиотека React.js. Сборка и отладка проекта производились на Vite. Сайты развернуты на облачной платформе Vercel.
